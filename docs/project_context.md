\# Project Context: SpotCheck

\*\*App Name:\*\* SpotCheck

\*\*Store Name:\*\* SpotCheck: Bill Defender

\*\*Version:\*\* 1.0 (Production Spec)

\*\*Target Market:\*\* UK (Primary)

\*\*Core Value:\*\* "Privacy-First Admin Automation."

\## 1. The Core Problems

\* \*\*The "Stupid Tax":\*\* Users forgetting renewal dates for MOTs, Passports, and Insurance.

\* \*\*The "Loyalty Penalty":\*\* Users overpaying for broadband/insurance because they don't know the market rate.

\* \*\*The "Passive Trap":\*\* Competitors (Emma, Snoop) only \*show\* spending; they don't help reduce it.

\## 2. The Solution: "Scan, Analyze, Vanish"

A React Native app using \*\*Local OCR\*\* for privacy and \*\*Cloud AI\*\* for intelligence.

\### Key Features (The "Killer" Suite):

\* \*\*Feature A: The "Loyalty Audit" (Insurance)\*\*

\* \*\*Logic:\*\* "Ghost Quote" comparison. User checks new customer price for \*themselves\*.

\* \*\*AI Role:\*\* If Renewal > Ghost Quote, generates "FCA GIPP Breach Complaint" script.

\* \*\*Feature B: The "Market Check" (Broadband/Mobile)\*\*

\* \*\*AI Role:\*\* Compares price vs hardcoded UK Averages. Generates negotiation script.

\* \*\*Feature C: The "Fine Print Assassin"\*\*

\* \*\*AI Role:\*\* Scans contracts for "Exit Clauses" (e.g., Moving house) and drafts cancellation letters.

\* \*\*Feature D: The "Warranty Resurrector"\*\*

\* \*\*AI Role:\*\* Drafts "Consumer Rights Act 2015" claim letters for broken goods >1 year old.
